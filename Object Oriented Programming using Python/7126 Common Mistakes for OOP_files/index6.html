Scenario

TypeError: object() takes no parameters

Possible Cause:

You might have written the constructor method name ("__init__") incorrectly. For example, missing one leading or trailing underscore from the constructor.

Example

  1. class Employee:
  2.     Emp_count=0
  3.     def _init__(self, name): 
  4.         self.__name = name
  5.         Employee.Emp_count += 1
  6. emp1 = Employee('Raj')
  7. print(emp1.Emp_count)

A leading underscore is missed resulting in "init" being treated as instance method rather than a constructor. So the initialization never happens for the instance variables.

How do I fix it?

Make sure that you write the correct syntax for the constructor

Reinforce your learning and practice on:

Writing methods with predefined syntax such as init.
