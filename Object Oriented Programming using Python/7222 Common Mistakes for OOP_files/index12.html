Scenario

AttributeError: 'function' object has no attribute 'XXX'

Possible Cause:

You might be trying to access local variable of a method from another method using methodname.variablename. Scope of the local variables is the method in which they are declared. They cannot be accessed outside the method.

Example

The get_discount() method returns discount based on the number of employees.

  1. class Employee:
  2.     __Emp_count=0
  3.     def __init__(self, name):
  4.         self.__name = name
  5.         Employee.__Emp_count += 1
  6.     
  7.     def calc_discount(self):
  8.         if(Employee.__Emp_count > 5):
  9.             discount = 0 
  10.         else:
  11.             discount = 10 
  12.     def calc_amount(self, txn_cost):
  13.         self.calc_discount()
  14.         return txn_cost * self.calc_discount().discount 
  15. emp = Employee("John")
  16. print(emp.calc_amount(100))

The instance method calc_amount is trying to access local variable discount that is defined in another instance method calc_discount in the same class. Local variables are destroyed after the method invocation and they cannot be accessed from another method.

How do I fix it?

To access a local value calculated in one method in other methods:

  1. Store calculations performed in a method in an instance variable to make that data available to other methods
  2. You can also return the calculated value from a method and call this method from other methods to get the calculated value
